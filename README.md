# rustxt — качество русского текста для Claude Code

Единый скилл: типографика, инфостиль, Anti-AI паттерны, UX-тексты, деловая переписка, скоринг.
Работает в Claude Code, OpenClaw, Cursor, Codex.

## Что делает

- **Типографика** — кавычки «ёлочки», тире, неразрывные пробелы, №, ₽ — автоматически
- **Инфостиль** — 97 стоп-слов, структура текста, числа и факты, 139 антипаттернов
- **Anti-AI** — 19 паттернов, которые выдают AI-генерацию: рубленые фразы, Rule of Three, synonym cycling, signposting, промо-язык
- **Грамматика и пунктуация** — согласование, капитализация, запятые, тавтология
- **UX-writing** — кнопки, ошибки, подсказки, формы, диалоги
- **Деловая переписка** — email, мессенджеры, тон, отказы
- **Скоринг** — оценка 0–10 по 5 измерениям: `/rustxt:ru-score`

## Пример

### До (AI-слоп)

> Давайте разберёмся, как построить продуктивный день. Знакомо? Вы открываете ноутбук утром понедельника и понимаете, что задач больше, чем часов в сутках.
>
> Проблема не в инструментах, а в подходе. Не просто планирование, а именно система.
>
> Наше решение **выступает в роли** персонального ассистента, который **обеспечивает** эффективность, продуктивность и баланс. Оно было **разработано** с учётом **лучших практик** тайм-менеджмента.
>
> - **Скорость:** Задачи создаются мгновенно
> - **Простота:** Интерфейс интуитивно понятен
> - **Надёжность:** Данные синхронизируются в реальном времени
>
> Несмотря на определённые вызовы, связанные с адаптацией пользователей, результаты обнадёживают. Готовы узнать больше?

### После (rustxt)

> В понедельник у меня было 14 задач и 6 часов до конца дня. К обеду осталось 3. Остальные оказались мусором, который я таскал из пятницы.
>
> Я перестал планировать в голове и перенёс всё в один список. Без категорий, без цветов. Просто список. Утром вычёркиваю неактуальное и работаю с тем, что осталось.
>
> Приложение помогает одним: не даёт задаче потеряться между заметками, чатами и почтой. Новая задача создаётся за 2 секунды, голосом или текстом.

### Что исправлено

- «Давайте разберёмся» — signposting (анонсирование вместо действия)
- «Знакомо?» / «Готовы узнать?» — AI-риторические вопросы
- «Понедельник, вы открываете ноутбук» — киношный заход
- «Не в инструментах, а в подходе» — ложная глубина
- «Не просто X, а именно Y» — AI-клише
- «Выступает в роли» — copula avoidance (вместо простого «это»)
- «Скорость, простота, надёжность» — Rule of Three
- **Жирные заголовки** в списке — inline-header bold
- «Несмотря на вызовы... обнадёживают» — шаблон «проблемы и перспективы»
- Пассивный залог, канцелярит, промо-язык — убраны

## Установка

```bash
git clone https://github.com/mr-SuperNew/rustxt.git ~/.claude/skills/rustxt
```

Готово. Скилл автоматически активируется при работе с русским текстом.

## Персонализация голоса

Скилл разделяет **движок** (правила) и **данные** (ваш стиль).

Движок работает из коробки. Чтобы подключить свой голос, запустите:

```
/rustxt:setup
```

Скилл спросит: есть ли у вас файл Tone of Voice. Если есть, попросит путь. Если нет, проведёт короткое интервью (5 вопросов) и создаст файл стиля за вас.

После настройки весь текст будет звучать как вы, а не как generic AI.

## Скоринг

Команда `/rustxt:ru-score` оценивает текст по 5 измерениям:

| Измерение | Вес |
|-----------|-----|
| Типографика | 0,15 |
| Чистота языка | 0,25 |
| Грамотность | 0,20 |
| Структура | 0,20 |
| Точность для читателя | 0,20 |

Результат: 0–10 с разбивкой по каждому измерению и конкретными замечаниями.

## Кредиты

- **ru-text engine** — [Arseniy Kamyshev](https://github.com/talkstream/ru-text). Типографика, инфостиль, антипаттерны, грамматика, UX, деловая, скоринг
- **Anti-AI patterns** — Ilya Guzhov. AI-специфичные клише и правила живого текста
- **Humanizer adaptations** — [blader](https://github.com/blader/humanizer), based on [Wikipedia: Signs of AI writing](https://en.wikipedia.org/wiki/Wikipedia:Signs_of_AI_writing)
