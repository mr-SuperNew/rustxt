# ru-text config — Voice Data Paths (Example)

This file connects the ru-text engine to personal voice/brand data.
Copy this file to `config.md` and fill in your paths.

## Voice Profile

| Data | Path |
|------|------|
| Tone of Voice | path/to/your/tov.md |
| Author Profile | path/to/your/profile.md |
| Brand Identity | path/to/your/brand.md |

## Usage

When writing content for this author:
1. Read ToV first — it defines the linguistic fingerprint
2. Read Author Profile — it provides context and communication style
3. Apply all ru-text rules through the lens of this specific voice
